# Create javalce-app

Create projects as Javier would.

## Installation

```bash
pnpm install -g javalce-app
```

## Usage

```bash
# Interactive mode
pnpm create javalce-app

# Override template
pnpm create javalce-app --template <template>

# Override name
pnpm create javalce-app --name <name>

# Get Command help
pnpm create javalce-app --help
```
